# Naukri Job Market Analysis Dashboard

## Project Objective

The objective of this project is to analyze job market trends using Naukri job postings data. The dashboard helps identify hiring patterns, in-demand skills, top hiring companies, job locations, and experience-level requirements using Power BI.

## Dataset Used

The dataset contains job postings from the Naukri platform, including information such as company name, job title, required skills, job location, and experience level. 🔗 [Dataset](https://github.com/Soujanya9380/Naukri-Job-Market-Analysis-Dashboard/blob/main/dataset.csv)

## Questions / KPIs

* What is the total number of job postings?
* Which companies are hiring the most?
* What are the top in-demand skills?
* Which locations have the highest number of job opportunities?
* What experience levels are most commonly required?  [View Dashboard](https://github.com/Soujanya9380/Naukri-Job-Market-Analysis-Dashboard/blob/main/dashboard.png)

## Process

1. Collected and loaded the dataset into Power BI.
2. Performed data cleaning and transformation.
3. Created calculated columns and measures.
4. Designed interactive visuals and filters.
5. Built an interactive dashboard for analysis.

## Conclusion

The dashboard provides useful insights into current job market trends. It helps understand which skills are in demand, where job opportunities are concentrated, and which companies are actively hiring. This analysis can help job seekers focus on the most relevant skills and locations for better career opportunities.
